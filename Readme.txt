1. Dezipper le dossier
2. Dans votre terminal, positionnez vous dans le dossier mère du dossier (notamment credit-risk-prediction) 
3. Installation de l'environnement virtuel python
3.a. Tapez la commande "python3 -m venv venv' pour creer l'environnement virtuel python
3.b. Tapez la commande "source venv/bin/activate" pour activer l'environnement virtuel python
3.c. Tapez la commande "pip install -r requirements.txt" pour l'installation des packages
4. Tapez  afin la commande "make run_all" pour lancer le programme
